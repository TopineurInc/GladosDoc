## Topineur

**Topineur** is a new functional programming language designed for high performance, simplicity, and ease of learning.

Built with **Haskell**, **Topineur** stands on a foundation of reliability, expressive abstraction, and mathematical precision. This strong base enables a robust compiler design, a safe and expressive type system, and a flexible architecture engineered for long-term scalability and innovation.

## Core Technologies

| Layer                    | Technology              | Purpose                                               |
| ------------------------ | ----------------------- | ----------------------------------------------------- |
| **Language Foundation**  | **Haskell**             | Core language for compiler and runtime implementation |
| **Parser & Syntax**      | **Parsec**              | Parsing and syntax analysis for Topineur source code  |
| **Code Generation**      | **LLVM Backend**        | Efficient machine code generation and optimization    |
| **Testing & Validation** | **HUnit**               | Testing framework for Haskell programs                |
| **Package Management**   | **Stack**               | Dependency management and build automation            |
| **Documentation Tools**  | **Fumadocs**            | Compiler and library documentation                    |
