## Overview

Welcome to **Topineur**! This guide will walk you through everything you need to start building with our language.

## Installation

<Steps>
  <Step>
Download the compiler :

```bash tab="<Terminal /> Using curl"
# On Linux/macOS
curl -sSL https://topineur.org/install.sh | bash
```

```bash tab="Using wget"
# On Linux/macOS
wget -qO- https://topineur.org/install.sh | bash
```
  </Step>
  <Step>
Verify installation :

```bash title="Bash"
glados --version
```
  </Step>
</Steps>

## Your first Topineur Program

Create a new file called `hello.top` and add the following code:

```topineur title="hello.top"
|- Main function

def main(): Int {
  print("Hello, Topineur!")
}
```

Run your program using:

```bash
glados hello.top
```

## Basic Syntax

### Variables and Types

In **Topineur**, you can use different types. For example :

```topineur
let name: String = "World"      |- String Variable
let age: Int = 25               |- Integer Variable
let height: Float = 1.75        |- Float Variable
let isActive: Bool = true       |- Boolean Variable
```

### Functions

Functions in Topineur are declared using the `def` keyword, and to return use the `top` keyword :

```topineur
|- Define a function add
def add(a: Int, b: Int) -> Int {
    top a + b
}

|- Concatenates strings
def print_variable(variable: String) {
    print("Variable is " ++ {variable} ++ "!")
}

|- Call the other functions
def main(): Int {
  let variable: Int = add(10, 5)

  print_variable(variable)
}
```

if score > 100

### Objects

Objects allow you to organize data and behavior together. Here's how we define an objects:

```topineur
|- Define an object Person
object type Person {
  name: String
  age: Int

  |- Introduce the person
  def introduce(): Void {
    print("Hi, I'm " ++ self.name ++ " and I'm " ++ self.age ++ " years old.")
  }

  |- Method with logic
  def is_adult(): Bool {
    if (self.age < 18)
      then top True
    else
      top False
  }
}

|- Use the Object
let laurent = Person {
  name = Laurent
  age = 20
}
laurent.introduce()
```
